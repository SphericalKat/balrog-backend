# Forward-proxy
A stateless forward proxy for intercepting http(s) traffic and acting as a gateway. This particular service is built in Go. Install the dependencies to get started.


## Dependencies:
 - [Go](https://golang.org/)
 - [Redis](https://redis.io)

## Run the server
```bash
go run .
```

## Build a binary
```shell
go build . -o <filename>
```
